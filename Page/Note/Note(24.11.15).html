Queue 문제

백준 2164번  문제가 너무 쉬워서. 30분 이내에 풀었다. 근데, 역시 내 컴퓨팅 사고가 많이 모자라긴 하나 보다. while문 내부에 조건문을 넣을 필요가 없는데, 조건 검사를 돌려서 한 번 실패했다. 찾아 봤을 때 바로 문제를 발견해서 다행이긴 하다.
백준 11286번  PriorityQueue와 Comparator 클래스를 몰라서 좀 헤맸다. PriorityQueue클래스는 선언 시 인자로 Comparator를 지정하여 비교 입력이 가능. Comparator를 커스텀 하여 잘 이용하면 필요한 순서로 큐에 자료 입력이 가능하다. Comparator는 음수 값을 반환 받을 경우 비교 값의 순서를 바꾸어 입력 한다는 점을 기억하자.
