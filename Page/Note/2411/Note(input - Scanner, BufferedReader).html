Java에서 외부 입력을 받을 때 대표적인 방법은 2가지.

1. Scanner


공백과 개행(' ', '\n, \t, \r 등')으로 구분되는 요소를 토큰(token)이라고 하는데, Scanner의 입력 메소드는 토큰을 기준으로 구분하여 읽어 들인다.


nextX()의 형태를 주로 이룸(대부분 공백 기준 구분). X자리에는 입력 받을 데이터 타입을 표시. 단, String 타입의 데이터를 받을 때는 nextString()이 아닌, next(): 공백 기준 구분 or nextLine(): 개행 기준 구분


hasNext()는 nextX()와 마찬가지로 X 자리에 데이터 타입을 표시함. 입력 받은 값(X와 동일 타입)이 있으면 true, 그렇지 않다면 false 반환.


findInX(A)은 입력받은 내용에서 A와 일치하는 내용이 있는지 확인하여 해당 키워드를 그대로 출력함.


사용 예시

		
import java.util.Scanner;

public class ScannerFindInLineExample
  {
    public static void main(String[] args)
      {
        Scanner scanner = new Scanner("Hello, this is a sample text with numbers 12345.");

        // "sample"이라는 단어를 찾기
        String match = scanner.findInLine("sample");
        if (match != null)
          {
            System.out.println("찾은 단어: " + match);
          } else
          {
            System.out.println("단어를 찾지 못했습니다.");
          }

        // 숫자 패턴을 찾기
        String numberMatch = scanner.findInLine("\\d+");
        if (numberMatch != null)
          {
            System.out.println("찾은 숫자: " + numberMatch);
          } else
          {
            System.out.println("숫자를 찾지 못했습니다.");
          }

        scanner.close();
      }
  }


2. BufferedReader


InputStreamReader(문자열을 입력 받으면 Character 단위로 읽음)에 버퍼링 기능이 추가된 클래스임.


문자나 배열, 라인들을 효율적으로 읽을 수 있음. Buffer 사이즈는 지정 가능.
입력이 있을 때마다 한 문자씩 버퍼로 전송하는데, 버퍼의 사이즈를 넘어서거나 개행문자가 나타나면 버퍼의 내용을 한번에 프로그램에 전달함.


다만, Scanner와는 달리 입력 받은 데이터가 String 자료형으로 고정된다. 그렇기 때문에 한 번 가공을 해야 하는 경우가 많지만, 한 번에 묶어서 입력을 처리하다 보니 Scanner보다 처리 속도가 빠르다.


예를 들어서, 읽어온 데이터를 공백 단위로 가공하고자 한다면, StringTokenizer나 String의 split 메서드를 사용한다.


BufferedWriter도 있다. 이름에서 알 수 있듯 출력할 때 사용하는데, Reader와 마찬가지로 버퍼를 통한 출력 방식이기 때문에 개행이 필요하다면 newLine() 메서드 등을 이용하여야 한다.

		
import java.io.BufferedReader;
import java.io.IOException;
import java.io.InputStreamReader;
import java.util.StringTokenizer;

public class Main
  {

    BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    // String
    String s = br.readLine();

    // int
    int num = Integer.parseInt(br.readLine());

    // 공백 구분 방법 1
    StringTokenizer st = new StringTokenizer(s); // StringTokenizer 인자 값에 입력 문자열이 들어감
    int a = Integer.parseInt(st.nextToken()); // 첫번째 호출
    int b = Integer.parseInt(st.nextToken()); // 두번째 호출

    // 공백 구분 방법 2
    String array[] = s.split(" "); // 공백마다 데이터 끊어서 배열에 넣음
    br.close(); // 입출력이 끝난 후 닫기
  }
    catch(IOException e)
        { // 예외처리 필수
          e.printStackTrace();
          System.out.println(e.getMessage());
        }
